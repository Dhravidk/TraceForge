# Demo Script

## Working version

This is the live demo script scratchpad for the JacHacks submission.

## Target arc

1. Open TraceForge and explain the problem in one sentence.
2. Load a sample batch of mini-SWE-agent trajectories.
3. Show top-level failure families and clusters.
4. Open one representative failed run.
5. Highlight the likely critical step.
6. Show a typed diagnosis and generated `AGENTS.md` patch.
7. Compare raw-baseline output against structured output.
8. Close on why Jac made the product possible.
